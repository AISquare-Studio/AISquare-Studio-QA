## How to Use

1. Open a new GitHub issue on the repository
2. Copy the contents of the desired issue file
3. Apply the labels listed in the issue front matter
4. Publish the issue

Contributors can then pick up issues and submit pull requests following the
[Contributing Guide](../../CONTRIBUTING.md).
