### 3. ETag Computation

```python
def compute_etag(metadata: Dict[str, Any], steps: List[str]) -> str:
    canonical_steps = [" ".join(s.strip().lower().split()) for s in steps]
    payload = {
        "flow_name": metadata.get("flow_name", ""),
        "tier": metadata.get("tier", ""),
        "area": metadata.get("area", ""),
        "steps": canonical_steps
    }
    hash_input = json.dumps(payload, sort_keys=True)
    return hashlib.sha256(hash_input.encode()).hexdigest()  # Full hash
```

**Purpose:** Idempotency - same metadata + steps = same ETag = no regeneration
